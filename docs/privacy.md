# プライバシーポリシー

最終更新：2025年12月11日

## 概要
MuscleMemoは、ユーザーのプライバシーを最優先に設計されています。

## データ収集について
**本アプリはいかなるデータも収集・送信しません。**

### 保存されるデータ
以下のデータはお使いのデバイス内にのみ保存されます：
- トレーニング記録（種目、重量、回数、日付）
- カスタム種目情報
- アプリ設定

### データの利用目的
- トレーニング履歴の表示
- 統計情報の計算
- 1RM推定の計算

### 第三者との共有
**一切行いません。** すべてのデータはデバイス内にのみ保存されます。

### データの削除
アプリをアンインストールすることで、すべてのデータが完全に削除されます。
